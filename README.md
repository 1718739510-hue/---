# Penguin Flight - A HarmonyOS Flight Management App

Penguin Flight is an independently developed flight management app based on HarmonyOS. It covers the full operation flow from user registration/login to flight search, reservation, order management, and cancellation, with special focus on user experience and system stability.

## Main Features
- Secure registration, login, and account management
- Flight search and detail viewing
- Flight booking, order management, and order cancellation
- Asynchronous data loading, permission checking, and smooth page routing
- Clear, modular code structure, easy to extend and maintain
- Pretty UI, responsive interaction, tested on real HarmonyOS devices
- Written in TypeScript, leveraging HarmonyOS Ability, Page, FA model, local storage APIs, and more

## Getting Started
1. Clone this repository and open it in the HarmonyOS IDE
2. Follow the README instructions to compile and deploy to a supported device or dev board
3. Register and explore flight booking flows—experience the complete business loop
